Front-end: CSS
Cor do texto

A propriedade color do CSS definine o valor da cor de um elemento com seu conteúdo em texto e seus decorações (estilos).

color: red
Peso da fonte

A propriedade CSS font-weight especifica o peso ou a intensidade da fonte (ex.: negrito). Algumas fontes oferecem apenas as opções normal e negrito.

font-weight: bold
Cor de fundo

A propriedade CSS background é um atalho para definir os valores de fundo individuais em um único lugar na folha de estilo.

background: blue
Centralizar texto

A propriedade CSS text-align descreve como conteúdo inline, como texto, é alinhado no elemento pai em bloco. text-align não controla o alinhamento de elementos em bloco, apenas o seu conteúdo inline.

text-align: center
